# Sf::Thinclient

一个gem包,用于ssl瘦客户机自动化测试

## Installation

Add this line to your application's Gemfile:

    gem 'sf-thinclient'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sf-thinclient
